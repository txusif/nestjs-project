### To create a controller using the CLI
```bash
nest g controller [name]
```

### To create a service using the CLI
```bash
nest g service [name]
```

### To create a module using the CLI
```bash
nest g module [name]
```

### To create everything with a single command
```bash
nest g resource <employees>
```
